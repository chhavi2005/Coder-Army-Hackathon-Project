Eye Controlled Virtual Mouse

Technologies: Flask & Python Libraries

The "Eye Controlled Virtual Mouse" is an innovative project designed to assist individuals with disabilities who are unable to use a traditional mouse. By leveraging the power of pyautogui, OpenCV, and MediaPipe, this project enables users to control mouse movements through eye tracking using a webcam.
